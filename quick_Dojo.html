  2. Odd Numbers 1-1000
  3. Print Sum of all odd numbers 1-5000
  4. Iterating Through the Array: Given an array X say [1,3,5,7,9,13], write a program that would iterate through each member of the array and print each value on the screen.
  5. Find Max: Given an array with multiple values (e.g. [-3, 3, 5, 7]), write a program that prints the maximum number in the array.
  6. Find Average: Given an array with multiple values (e.g. [1,3,5,7,20]), write a program that prints the average of the values in the array.
  7. Array with Odd Numbers: Write a program that creates an array 'Y' that contains all the odd numbers between 1 to 255.
  8. Greater than Y: Write a program that takes an array and returns the number of values in that array whose value is greater than y
  9. Square the Values: Given an array x (e.g. [1,5, 10, -2]), create an algorithm that squares each value in the array.
  10. Eliminate Negative Numbers: Given an array x (e.g. [1,5, 10, -2]), create an algorithm that replaces any negative number with the value of 0
  11. Max, Min, Average: Given an array x (e.g. [1,5, 10, -2]), create an algorithm that prints the maximum, minimum, and average values in the array.
  12. Shifting the values in the array: Given an array x (e.g. [1,5, 10, 7, -2]), create an algorithm that shifts each number by one (to the front).
  13. Number to String: Write a program that takes an array of numbers and replaces any number that's negative to a string called 'Dojo'.
  14. Random Array: Create an array X and fill the array with 10 values, each value being a random integer between 0 to 100
  15. Swapping two values: array of numbers and returns an array where the first and last numbers in the array have been switched
  16. Reversing: Given an array X of multiple values (e.g. [-3,5,1,3,2,10]), write a program that reverses the values in the array.
  17. Insert X in Y: Write a program that inserts a new number X at an index Y.
  18. Removing Negatives: Given an array of multiple values (e.g. [0, -1, 2, -3, 4, -5, 6]), write a program that removes any negative values in that array.
